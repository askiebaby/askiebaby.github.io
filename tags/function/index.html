1. array

複數

  • 像火車車廂的概念。
  • 有順序性，可以放各種資料，但建議資料類型相似的一組資料結構/格式。
  • 適合資料處理。
  • 表達複數的概念，多個相同的複類別。
  • 可以放不一樣型別的值，而強型別語言中都會相同型別的值。（2019-7-16 更新）
