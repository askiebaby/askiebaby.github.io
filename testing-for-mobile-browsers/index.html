方法（2019-07-16 更新）

  1. 在 iTerm 進入專案的資料夾根目錄。

  2. 找到對外 IP，要找 IPV4 的 。

ifconfig // MacOS
ipconfig // Windows
  1. 輸入以下指令，預設 port 是 8000。
python -m SimpleHTTPServer [port]
  1. 在瀏覽器輸入 IP:port，直接對外測試。
